## 📊 详细变更分析

### 🏗️ 组件 1：场景定义注册表（SceneDefRegistry）

#### 新增文件：`scripts/SceneDefRegistry.js`

**变更说明**：
- 实现了三层场景定义获取机制：硬编码缓存 → JSON 缓存 → 同步查表
- **核心函数**：
  - `registerScene(id, def)` - 注册硬编码场景
  - `getSceneDefSync(id)` - 同步获取（用于检查点恢复等同步上下文）
  - `resolveSceneDef(id)` - 异步解析（优先硬编码，否则 fetch JSON 并缓存）

**技术亮点**：
- 双缓存设计：`_hardcoded` 存储硬编码场景，`_cache` 存储 JSON 场景
- 懒加载：JSON 场景按需加载并缓存
- 兼容性：支持新旧场景并存（硬编码 + 外部化）

---

### 🏗️ 组件 2：游戏入口解耦（Game & Character Demo）

#### 文件：`character_demo.js`

| 变更项 | 旧代码 | 新代码 | 说明 |
|--------|--------|--------|------|
| 导入 | `import { OUTDOOR_VILLAGE, BATTLE_DEFS }` | `import { Game }` | 移除硬编码场景依赖 |
| 初始化调用 | `await game.init(OUTDOOR_VILLAGE, BATTLE_DEFS)` | `await game.init()` | 改为无参数，由内部决定起点 |

#### 文件：`scripts/Game.js`

**变更说明**：
1. **导入变更**：
   - ❌ 移除：`import { ALL_SCENES, OUTDOOR_VILLAGE }`
   - ✅ 新增：`import { resolveSceneDef, getSceneDefSync }`

2. **init() 方法重构**：
   ```javascript
   // 旧：async init(sceneDef, battleDefs)
   // 新：async init()
   async init() {
       const sceneId = this.worldState.currentSceneId;
       const sceneDef = await resolveSceneDef(sceneId);
       await this.scene.init(sceneDef, BATTLE_DEFS);
   }
   ```

3. **resetWorldState() 增强**：
   - 新增：`this.worldState.currentSceneId = "prologue"`
   - 新增：`this.worldState.currentSpawnId = null`

4. **questManager.onStateChange 优化**：
   - SpawnId 优先级：`worldState.currentSpawnId` → 场景第一个 → "house_door" 兜底

5. **restoreCheckpoint() 同步化**：
   - 旧：`ALL_SCENES[cp.sceneId]`
   - 新：`getSceneDefSync(cp.sceneId)`

### 🏗️ 组件 3：场景初始化容错（Scene）

#### 文件：`scripts/Scene.js`

**变更说明**：
移除强制依赖 `battleDefs["battle_field_1"]` 的兜底逻辑，改为使用默认配置：

| 配置项 | 默认值 | 用途 |
|--------|--------|------|
| `DEFAULT_DUEL_CAMERA` | `zoomMinDistance: 3.2`, `zoomMaxDistance: 6.4`, `targetAspect: 16/9` 等 | 无战斗场景的相机配置 |
| `DEFAULT_STAGE_BOUNDS` | `minX: -8, maxX: 8, minY: -0.05, maxY: 0.05` | 无战斗场景的舞台边界 |

**逻辑流程**：
```javascript
const firstBattleTrigger = sceneDef.triggers?.find(t => t.type === "battle");

if (firstBattleTrigger) {
    // 有战斗：使用 battleDefs 配置
    const battleDef = battleDefs[firstBattleTrigger.battleId];
    stageBounds = battleDef.stageBounds;
    duelCameraCfg = battleDef.duelCamera;
} else {
    // 无战斗：使用默认配置
    stageBounds = DEFAULT_STAGE_BOUNDS;
    duelCameraCfg = DEFAULT_DUEL_CAMERA;
}
```
